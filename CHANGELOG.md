# Changelog

## [1.0.0] - 2026-02-14

### Features
- 34 Organisationen (Sektionen, Bereiche, Teilbereiche)
- 3 Kategorien: Freie Hochschule, Anthroposophische Gesellschaft, Sonstige
- 2 Layouts: Desktop (horizontal), Mobile (vertikal)
- 4 Sprachen: DE, EN, FR, IT
- 5 Farbvarianten: Original, Weiß, Schwarz, Grau, Custom
- Custom Colors: Icon/Zeile 2, Zeile 1
- 3 Hintergrund-Optionen: Weiß, Schwarz, Transparent
- Größenregler: 1x - 6x
- Export: SVG, PNG, JPG
- Text-to-Path Konvertierung für konsistente SVG-Exports
- Responsive 3-Spalten Layout
- Live-Preview mit dynamischer Größenanpassung

### Technical
- Pure HTML/CSS/JavaScript (keine Dependencies)
- Embedded Font: TitilliumUpright Regular (Base64)
- 38 vollständige Glyphen für DE/EN Texte
- Dynamic viewBox berechnung mit getBBox
- Letter-spacing: -0.01em, Word-spacing: -0.07em
