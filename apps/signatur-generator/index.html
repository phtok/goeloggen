So wird daraus eine Signatur

      
Zwei Wege, ein Ergebnis. In den meisten Fällen genügt Kopieren.

    


    

      

        
Normalfall

        
‹Signatur kopieren›

        
Für neues Outlook, Outlook (Mac), Apple Mail und Gmail. Legt die fertige,
          formatierte Signatur in die Zwischenablage; du fügst sie im Signatur-Editor des
          Programms ein (Strg/Cmd + V). Nichts wird verschickt, alles bleibt lokal.

      

      

        
Nur klassisches Outlook (Windows)

        
‹.htm laden›

        
Lädt eine Datei für den Outlook-Signaturordner – nützlich, wenn das Einfügen die
          Formatierung verliert. Achtung: Das neue Outlook und Apple Mail/Gmail nutzen
          diese Datei nicht; dort einfach kopieren.

      

    


    

      
Menübezeichnungen ändern sich je nach Version – bei
        Abweichungen gilt die jeweils verlinkte Hersteller-Anleitung.


      

        Neues Outlook – Windows kopieren
        
    
          
  1. ‹Signatur kopieren› klicken. (Neues Outlook erkennst du am Schalter ‹Neues Outlook› oben rechts.)
    2. 
          
  2. Einstellungen (Zahnrad) → Konten → Signaturen.
    3. 
          
  3. ‹Neue Signatur›, Namen vergeben, ins Feld einfügen (Strg + V), speichern.
    4. 
          
  4. Als Standard für neue Mails und Antworten wählen.
    5. 
        

        
Das neue Outlook nutzt nicht den
          %APPDATA%-Ordner – ‹.htm laden› funktioniert hier nicht.
          Offizielle Anleitung:
          Microsoft Support.

      


      

        Klassisches Outlook – Windows kopieren oder ‹.htm›
        
    
          
  1. Variante kopieren: neue Mail → Menü ‹Nachricht› → ‹Signatur› →
            ‹Signaturen…› → ‹Neu› → einfügen (Strg + V).
    2. 
          
  2. Variante Datei: ‹.htm laden›; Explorer öffnen, in die Adresszeile
            %APPDATA%\Microsoft\Signatures eingeben, Datei dort ablegen.
    3. 
          
  3. Outlook → Datei → Optionen → E-Mail → ‹Signaturen…› – Signatur erscheint, als
            Standard zuweisen.
    4. 
        

        
Offizielle Anleitung:
          Microsoft Support.

      


      

        Outlook – Mac kopieren
        
    
          
  1. ‹Signatur kopieren› klicken.
    2. 
          
  2. Outlook → Einstellungen → Signaturen → ‹+›.
    3. 
          
  3. In das Bearbeitungsfeld einfügen (Cmd + V), Konto zuweisen.
    4. 
        

        
Offizielle Anleitung:
          Microsoft Support.

      


      

        Apple Mail – Mac kopieren
        
    
          
  1. ‹Signatur kopieren› klicken.
    2. 
          
  2. Mail → Einstellungen → Signaturen → Konto wählen → ‹+›.
    3. 
          
  3. Wichtig: Häkchen ‹Standardschriftart immer verwenden› entfernen –
            sonst geht die Formatierung verloren.
    4. 
          
  4. In das rechte Feld einfügen (Cmd + V).
    5. 
        

        
Offizielle Anleitung:
          Apple Support.

      


      

        Gmail – Web kopieren
        
    
          
  1. ‹Signatur kopieren› klicken.
    2. 
          
  2. Zahnrad → ‹Alle Einstellungen anzeigen› → Tab ‹Allgemein› → Abschnitt ‹Signatur›.
    3. 
          
  3. ‹Neu erstellen›, Namen vergeben, ins Feld einfügen (Strg/Cmd + V).
    4. 
          
  4. Ganz unten ‹Änderungen speichern›.
    5. 
        

        
Offizielle Anleitung:
          Google Support.

      


      

        Handy – iOS / Android Hinweis
        
Mobile Mail-Apps zeigen formatierte Signaturen kaum zuverlässig.
          Empfehlung: die Kurz-Variante als einfachen Text in der App-Signatur eintragen –
          oder die Desktop-Signatur über das Konto synchronisieren lassen.
