QR-Code-Generator

QR-Codes im Hausstil – für Links, WLAN-Zugänge, Kontakte, E-Mail und Text. Link-Codes laufen auf Wunsch als Kurzlink und zählen ihre Scans: anonym und ohne Fremddienst.

Inhalt

Was der Code beim Scannen auslösen soll. Nur Link-Codes können zählen – alles andere steht fest im Code selbst.

Kurzname · tools.goetheanum.ch/s/… Erst Kurzlink anlegen verdrahtet den Code – dann zeigt der QR auf den Kurzlink und jeder Scan wird gezählt.

Gestalt und Ausgabe

Drei Stile, drei Farben – immer dunkel auf Weiss, damit jede Kamera den Code liest. SVG für Druck und Layout, PNG für Mail und Web.
