Zwei Wege, die Schrift zu laden

A · Von der Hausadresse laden. Am schnellsten: die Dateien direkt von werkzeuge.goetheanum.ch laden – über https, von jeder Domain nutzbar. Der Code steht unten.

B · Selbst hosten für Produktion empfohlen
Die .woff2 aus dem Download-Paket (Ordner Webfonts/woff2/) ins eigene Projekt legen – etwa /fonts/ – und die src-URL relativ setzen: url("/fonts/Goetheanum-Variabel-v2.7.woff2"). Alles aus einer Herkunft, die robusteste Wahl für eine echte Website.

1 · Variable Font einbinden empfohlen — eine Datei, 66 KB, alle Schnitte

@font-face {
  font-family: "Goetheanum";
  /* A · Hausadresse — oder B: url("/fonts/Goetheanum-Variabel-v2.7.woff2") */
  src: url("https://werkzeuge.goetheanum.ch/assets/fonts/goetheanum/Webfonts/woff2/Goetheanum-Variabel-v2.7.woff2") format("woff2"),
       url("https://werkzeuge.goetheanum.ch/assets/fonts/goetheanum/Webfonts/woff/Goetheanum-Variabel-v2.7.woff") format("woff");
  font-weight: 190 725;     /* variable Gewichtsachse */
  font-style: normal;
  font-display: swap;
}

/* Verwendung */
body { font-family: "Goetheanum", system-ui, sans-serif; }
.leise { font-weight: 265; }
.ruhig { font-weight: 350; }   /* Lesetext */
.klar  { font-weight: 440; }   /* Standard */
.laut  { font-weight: 680; }
.frei  { font-variation-settings: "wght" 520; }  /* stufenlos */

Schnitte: Flüstern 190 · Leise 265 · Ruhig 350 · Klar 440 · Deutlich 580 · Laut 680 · Schreien 725. Dazwischen alles stufenlos über font-variation-settings:"wght" ….

2 · Einzelne Schnitte (statisch)

@font-face{font-family:"Goetheanum Klar";
  src:url("https://werkzeuge.goetheanum.ch/assets/fonts/goetheanum/Webfonts/woff2/Goetheanum-Schrift-v2.7-Klar.woff2") format("woff2");
  font-weight:400;font-display:swap}
/* B (selbst hosten): src:url("/fonts/Goetheanum-Schrift-v2.7-Klar.woff2") format("woff2"); */
/* ebenso: …-Leise.woff2 als "Goetheanum Leise", …-Ruhig.woff2 als "Goetheanum Ruhig", …-Laut.woff2 als "Goetheanum Laut" */

Nutzung: font-family:"Goetheanum Klar". Verfügbar: Leise · Ruhig · Klar · Deutlich · Laut.

4 · Icons & Pfeile zwei eigene Fonts, per Taste

Piktogramme und Pfeile liegen in zwei eigenen Fonts – nicht im Textfont. Man setzt sie nicht über ein Gewicht, sondern indem man einer Fläche die Icon- oder Pfeile-Schrift gibt und die passende Taste tippt. Welche Taste welches Zeichen ergibt (und jedes einzeln als SVG/PNG/PDF), zeigt die Icon-Seite.

@font-face{ font-family:"Goetheanum Icons";
  src:url("https://werkzeuge.goetheanum.ch/assets/fonts/goetheanum/Webfonts/woff2/Goetheanum-Icons-v2.7.woff2") format("woff2"); font-display:block }
@font-face{ font-family:"Goetheanum Pfeile";
  src:url("https://werkzeuge.goetheanum.ch/assets/fonts/goetheanum/Webfonts/woff2/Goetheanum-Pfeile-v2.7.woff2") format("woff2"); font-display:block }

/* Verwendung – Zeichen per Taste, nicht per Gewicht */
.icon  { font-family:"Goetheanum Icons"; }   /* <span class="icon">h</span>  → Piktogramm */
.pfeil { font-family:"Goetheanum Pfeile"; }  /* <span class="pfeil">u</span> → Pfeil (Grossbuchstabe = fett) */

Beide Wege gelten auch hier: von der Hausadresse laden (oben) oder selbst hosten – Dateien ins Projekt, src-URL relativ.
