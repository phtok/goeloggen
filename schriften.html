Im Web einbinden.

Eine Datei, alle sechs Schnitte – der Variable Font ist nur 66 KB. Den @font-face-Block kopieren, fertig.

@font-face{
  font-family:"Goetheanum";
  src:url("https://phtok.github.io/goeloggen/assets/fonts/goetheanum/Webfonts/woff2/Goetheanum-Variabel-v2.6.woff2") format("woff2"),
      url("https://phtok.github.io/goeloggen/assets/fonts/goetheanum/Webfonts/woff/Goetheanum-Variabel-v2.6.woff") format("woff");
  font-weight:190 725; font-style:normal; font-display:swap;
}
/* Verwendung */
body{ font-family:"Goetheanum", system-ui, sans-serif; }
.leise{ font-weight:265 } .klar{ font-weight:440 } .deutlich{ font-weight:580 } .laut{ font-weight:680 }
.frei{ font-variation-settings:"wght" 520 }  /* stufenlos */
