Befund: vorher → nachher

Gerechnet am Original (Juli 2026) — Regel-IDs in Klammern.

Gold #ebb565 + Weiss 1.85:1 → Gold-Pille 4.55:1 (B01/B02: Auswahl = dunkles Gold + Weiss).

Kalender-UI 12.5–13px → ≥14/15px (B03); Grautöne 2.15–3.1:1 → --muted 4.9:1 (B02).

15× outline:none → sichtbarer Fokus, Skip-Link aus nav.js (WCAG 2.4.1/2.4.7).

74 Hex-Literale + CMS-Inline-Gradients → Tokens (DS02); damit wird Hell/Dunkel überhaupt erst möglich (B05).

11 Schnitt-Dateien als Fake-Familien → ein Variable Font; Titillium-Titel → Hausschrift (die Stimme).

Zeilenhöhe 1.4 (leading-normal) → 1.66 im Lesetext (B04).
