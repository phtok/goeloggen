Befund: vorher → nachher

Gerechnet am Original (Juli 2026) — Regel-IDs in Klammern.

Grau #6e6e6e auf #121212 = 3.67:1 → --stage-muted 8.2:1 (B02, gerechnet statt geschätzt).

Titillium per !important über Inter → Hausschrift als Quelle; VERSAL-Knöpfe → normale Beschriftung (G05/G23).

Schiefer #23323F ohne Marke → Blau/Gold-Tokens; Licht/Dunkel als duplizierte Literale → eine Token-Schicht (DS02/B05).

Kein :focus-visible, kein Skip-Link → Fokus-Defaults + Sprunglink aus base.css/nav.js (WCAG 2.4.1/2.4.7).

Google Fonts + vier Fremd-CDNs → Selbst-Hosting (Datenschutz, Reproduzierbarkeit).

Gelernt in Gegenrichtung: die Reihen-Kacheln und die abgeleiteten Töne per Formel — beides ist jetzt Teil des Fundaments (--stage-*).

Der Weg dorthin

Uscreen erlaubt eigenes CSS und eigene Head-Snippets — derselbe Kanal, der heute die !important-Flicken trägt.

<!-- Uscreen · Theme-Einstellungen · Custom Code (Head) -->
<link rel="stylesheet" href="https://werkzeuge.goetheanum.ch/design-system/tokens.css">
<link rel="stylesheet" href="https://werkzeuge.goetheanum.ch/design-system/base.css">
<style>
  /* Plattform-Variablen auf die Tokens heben – statt Schrift per !important zu flicken */
  :root{ --ds-fg-default:var(--stage-ink); --ds-fg-muted:var(--stage-muted) }
  body{ font-family:var(--font) }
</style>
